# lotto-checker(簡易大樂透兌獎工具)

## 使用說明

1. 根據當期的號碼, 特別號, 自己購買的樂透選號, 本期加開的樂透號修改`config/index.js`
2. cd `lotto-checker`
3. 執行本程式`node main.js`並印出中獎結果
4. 輸出結果如下

```
大樂透,沒中,中0碼,我的號碼:[2,9,12,20,42,44]
大樂透,沒中,中1碼,我的號碼:[4,16,26,31,43,48]
活動加碼,沒中,中1碼,我的號碼:[2,9,12,20,42,44],加開號碼:[1,2,5,31,35,39]
活動加碼,沒中,中2碼,我的號碼:[2,9,12,20,42,44],加開號碼:[1,2,7,9,30,47]
活動加碼,沒中,中1碼,我的號碼:[4,16,26,31,43,48],加開號碼:[1,2,5,31,35,39]
活動加碼,沒中,中0碼,我的號碼:[4,16,26,31,43,48],加開號碼:[1,2,7,9,30,47]
```
